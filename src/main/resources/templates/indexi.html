Aplicacion de Prueba

El proposito de esta aplicación es demostar mis conocimientos y habilidades, sin embargo derivado de la propia naturaleza de aplicación no es posible colocar de manera productiva el API de spotify, ya que necesita pasar por una auditoria por parte de spotify.

Sin embargo para demostar la funcionalidad de la api se colocan 2 usuarios que estan registrados como Testers , para que puedan ser usados.

¿Como iniciar sesión?

Para iniciar sesión es necesario colocar alguno de los correos provistos con anterioridad.

Posteriormete es necesario dar click en "Iniciar sesión con contraseña", y continuar con el login
